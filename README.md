# cmath

A core math package for golang, with statistics, histogram, points and ranges data structures and functions.
## Usage

### Points

- Creating two points. An int point and a float point.

```go
p2 := NewPoint(2, 2)
p3 := NewPoint(2.5, 3.775)
```

- Adding two points

```go
result := p1.Sum(p2)
```
